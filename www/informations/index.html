変更点は以下の通りです。
https://www.seasar.org/issues/browse/MOBYLET/fixforversion/10954 

Release Notes - Mobylet - Version 0.8.0


** Bug
    * [MOBYLET-22] - [mobylet-core] GoogleMapが表示されない
    * [MOBYLET-23] - [mobylet-core] DocomoでGps情報取得Aタグが正確に出力されない

Release Notes 0.7.0

2009/07/08 Ver.0.7.0をリリースしました。
変更点は以下の通りです。
https://www.seasar.org/issues/browse/MOBYLET/fixforversion/10958 

Release Notes - Mobylet - Version 0.7.0


** Bug
    * [MOBYLET-11] - [mobylet-core] Mobylet#getDisplay()で返却する情報の優先度を変更する
    * [MOBYLET-12] - [mobylet-core] 端末プロファイル情報を読み込み処理をsynchronizedする
    * [MOBYLET-14] - [mobylet-gaeextension] 画像リサイズ機能が動作しない
    * [MOBYLET-20] - [mobylet-charset] 句読点が文字化けする
    * [MOBYLET-21] - [mobylet-taglibs] imgタグがDynamicAttribute対応になっていない

** Improvement
    * [MOBYLET-16] - [mobylet-core] 画像キャッシュ構造を拡充する
    * [MOBYLET-17] - [mobylet-core] 画像変換処理部にImageMagickなどをキックできる
                                    I/F構造を作る

** New Feature
    * [MOBYLET-6] - [mobylet-core] 画像オートリサイズ機能の画像キャッシュ化対応
    * [MOBYLET-9] - [mobylet-taglibs] taglibの設定を外部ファイルに切り出すことが可能に
                                      拡張する
    * [MOBYLET-13] - [mobylet-taglibs] FormTagを作成する（ATag的機能を搭載）
    * [MOBYLET-15] - [mobylet-core] View側ロジックをcoreへ寄せる
    * [MOBYLET-18] - [mobylet-core] 平面移動Gps座標計算ユーティリティを作る
    * [MOBYLET-19] - [mobylet-core] 画像変換パターンを追加
